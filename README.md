# ROAD_ACCIDENT_CASUALTIES
 *Project Objective:*
1. Calculate the total casualties
2. Analyze year-wise casualties
3. Analyze casualties by road type
4. Analyze casualties by road surface
5. Analyze casualties by location
6. Analyze casualties by light condition
7. Analyze the number of casualties by vehicle type
8. Calculate the total number of fatal casualties
9. Calculate the total number of serious casualties
10. Calculate the total number of car casualties

*Project Steps:*
1. Gather requirements from the client
2. Identify project stakeholders
3. Clean data as per requirements
4. Process data by adding customized columns
5. Perform data analysis using Pivot Tables and Excel functions
6. Visualize data through charts and custom sheets to show insights
7. Create a report/dashboard from start to end
